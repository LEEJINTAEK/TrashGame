# TrashGame
슈팅 게임
## 언어
**JavaScript**<br>

## 설명
1. '스페이스 바'를 눌러 '총알'을 발사하세요.
2. '적'을 맞추면 Score가 올라갑니다.
3. Score 20점 도달시 적 속도가 2배 빨라집니다.
4. 적이 끝에 도달시 게임은 끝납니다.
